# Restaurant-Menu-Website
📌 Overview
Parvez Da Dhaba is a responsive restaurant menu website built using HTML and CSS. It showcases popular food items like Gulab Jamun and Rasgulla with their ingredients and recipes in a clean and user-friendly layout.

🚀 Features
Responsive design for mobile, tablet, and desktop

Clean and structured UI layout

Interactive food cards with hover overlay effect

Navigation bar with Home, About, and Contact links

Organized sections for ingredients and recipes

🛠️ Tech Stack
HTML5

CSS3

📂 Project Structure
index.html – Main structure of the website

style.css – Styling and layout design

images/ – Contains food images

💡 Functionality
Displays food items with details like ingredients and recipes

Hover effect on images shows "Click here to buy" overlay

Easy navigation through header links

🎯 Purpose
This project was developed as part of web development practice to improve skills in HTML, CSS, and responsive design.
